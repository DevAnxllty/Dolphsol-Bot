
# Dolphsol Bot

A discord bot utility for the Dolphsol Macro in Sol's RNG
## Features

- Multiple Commands
  - Restart
  - Inventory {Gears/Items}
  - Screenshot
- Control your Dolphsol Macro, away from your computer
- Easy to use
- Protection against other people messing around with your macro
- More coming soon..
